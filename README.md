
# Booxtract #

Это небольшой проект на Symfony, созданный в личных целях.  
Цель: наведение порядка в электронных книгах.  
Консольная утилита анализирует файлы с электронными книгами в указанной папке, собирает метаданные (автор, название, издание, год выпуска) и переименовывает файлы в соответствии с выбранным форматом названия.  

## Требования ##

Установленный в системе [Docker](https://www.docker.com)


## Текущие ограничения ##

На данный момент поддерживается только формат fb2.  
В будущем, планируется как минимум поддержка epub и mobi.    
Также сейчас поддерживается только один формат наименования:  

```
[%язык_если_не_русский%] %Автор% - %Название% (%издание_если_есть%) [%год%].%расширение%
```

## Установка и запуск ##

1. Склонировать проект.
2. (Опционально) Настроить файл `.env`: скопировать с шаблона `cp .env.dist .env` и изменить по необходимости
3. Поместить файлы книг в директорию `./books` (или любую другую директорию **внутри** проекта, указанную в опции `BOOK_DIR` в файле `.env`)
4. Запустить одну из команд: 

	# Проанализировать и переименовать (все автоматически)
	make process

	# Проанализировать и переименовать (ручной режим, будет спрашивать о каждом файле)
	make manual

	# Проанализировать и вывести в консоль предлагаемые названия
	make dry-run

	# Проанализировать и вывести в консоль все метаданные
	make analyze
